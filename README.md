This project presents an interactive Tableau dashboard for exploring urban air quality using multi-source datasets. It combines environmental, traffic, demographic, and health-related data to identify pollution trends and community vulnerability patterns.

## Project Overview

The dashboard helps answer questions such as:
- How do air quality readings vary across stations and time?
- What relationships exist between weather, traffic volume, and pollution?
- Which communities may face higher environmental and health vulnerability?
